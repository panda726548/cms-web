# cms-web

#### 介绍
cms系统
http://localhost/cms-web/login
项目启动后用户名密码：admin/admin123

#### 网站所使用的技术
- Maven分模块管理
- Spring+Hibernate4+SpringMVC
- 页面采用Freemarker
- Js库使用JQuery
- 树Ztree
- 上传文件,uplodify
- Ajax,Dwr
#### Maven的分模块管理
- basic-hibernate
- cms-core-->model,dao
- cms-user-->user,role,group
- cms-article-->channel,article
- cms-system-->link,backup,info
- cms-web-->web
- cms-parent
